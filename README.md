## Running the code
- Import the code inside [`src`](src/) into an [Eclipse](https://eclipseide.org/) (an IDE) project.
- Create a separate folder called `res` and import files inside the [`res`](res/) folder.
- Run [`AppFrame.java`](src/app/AppFrame.java).
